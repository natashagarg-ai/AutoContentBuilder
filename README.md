# 📝 Auto Content Builder (Version 1)

## Overview

Auto Content Builder is an AI-powered content generation workflow built using **n8n**. It automates the creation of social media content by generating post ideas, captions, hashtags, and detailed image prompts that users can use with their preferred AI image generation service.

The generated content is automatically stored in Google Sheets, making it easy to organize, review, and reuse.

---

## Features

- AI-generated social media content
- AI-generated image prompts
- Google Sheets integration
- Automated workflow using n8n
- Customizable prompt generation
- Supports multiple content topics

---

## Workflow

1. User provides a topic.
2. AI generates social media content.
3. AI creates a detailed image prompt.
4. Generated content is saved in Google Sheets.
5. Users can use the image prompt with any compatible AI image generation service.

---

## Tech Stack

- n8n
- Google Gemini
- Google Sheets
- REST APIs

---

## Current Limitations

- Generates image prompts only; image generation requires the user to use their own AI image generation service or API key.
- Uses Google Sheets for content storage.
- Does not schedule or publish content automatically.
